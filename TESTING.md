# اختبار Trae Agent محلياً

## التشغيل السريع

```bash
# تشغيل الخادم المحلي
pnpm dev

# أو
node test-local.js
```

## تعليمات الاختبار

1. **تشغيل الخادم**: 
   ```bash
   pnpm dev
   ```

2. **فتح المتصفح**: 
   اذهب إلى `http://localhost:3000`

3. **إعداد API**:
   - اختر مزود الخدمة (OpenAI, Anthropic, OpenRouter)
   - أدخل أي مفتاح API (للاختبار فقط)
   - اختر النموذج المطلوب

4. **اختبار المهام**:
   - اكتب مهمة برمجية مثل: "إنشاء سكريبت Python لحساب فيبوناتشي"
   - اضغط "إرسال"
   - ستظهر استجابة محاكاة من الخادم المحلي

## المميزات المدعومة في الاختبار

- ✅ واجهة المستخدم الكاملة
- ✅ إرسال الطلبات إلى API
- ✅ استقبال الردود
- ✅ عرض النتائج والملفات
- ✅ الإشعارات والحالات
- ✅ حفظ الإعدادات

## استكشاف الأخطاء

### المنفذ مشغول
```bash
# تغيير المنفذ في test-local.js
const PORT = 3001; // بدلاً من 3000
```

### مشاكل في الملفات
```bash
# التأكد من وجود الملفات
ls -la public/
```

### مشاكل في Node.js
```bash
# التحقق من إصدار Node.js
node --version

# يجب أن يكون 18+ أو أحدث
```

## الانتقال للإنتاج

بعد التأكد من عمل التطبيق محلياً:

```bash
# بناء المشروع
pnpm build

# النشر على Cloudflare Pages
pnpm deploy
```

## ملاحظات مهمة

- الخادم المحلي يستخدم محاكاة للـ API
- لا يتم إرسال طلبات حقيقية إلى مزودي الخدمة
- يمكن تعديل الاستجابات في `test-local.js`
- للاختبار الحقيقي، استخدم Cloudflare Functions
